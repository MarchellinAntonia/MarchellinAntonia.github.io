Final Project - Maximum Entropy Method for Named Entity Recognition (case study: food photos caption on instagram)

this final project is an reseach to apply the maximum entropy method to the process of natural language processing, NER (Named Entity Recognition). Data were taken from the food photo caption on instagram, the system will execute NER Process by applying the maximum entropy method based on weight calculation that is done by GIS (Generalized Iterative Scaling) algoritm. GIS algorithm will calculate the weight of each feature, maximize weight, and gain convergent by counting the number of features that match with the data being tested. The result of this application is based on the entity grouping tag: restaurant, location, varity, price, comment

  • IDE: Netbeans
  • Operating System: Windows
  • Date: June 2016
  • Service: Application Development
